Fields

  Name Description
AttachedProperty
The PropertyInfo this is attached to, or null. If this is null, it will be filled in when [!:GetDataProcessor] or Read(PropertyInfo, Object, Package, BinaryReader, Int64) is called.
ExclusiveToGames
Get the collection of PackageGame objects that this property is present for. If this is empty, then it includes all games.
Processor
Delegate to use to process a property value, or null to use a default. If this is null, it will be filled in when GetLocalNativeProperties(Type) or GetNativeProperties(Type) is called.
