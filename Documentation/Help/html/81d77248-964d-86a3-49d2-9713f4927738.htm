ModelMaterial Properties
ModelMaterial Class See Also Send Feedback

The ModelMaterial type exposes the following members.

Properties

  Name Description
AmbientColor
Get or set the ambient color of the material, where X is red, Y is green, Z is blue, and 1.0 for each is "brightest". The default is (0, 0, 0).
DiffuseColor
Get or set the diffuse color of the material, where X is red, Y is green, Z is blue, and 1.0 for each is "brightest". The default is (1, 1, 1).
DiffuseMap
Get or set the diffuse map, or null if there is none.
DiffuseTexture
Glossiness
Get or set the shininess of the material, which is the exponent used for specular reflections. Higher values produce a sharper reflection. The default is 10.
Name
Get or set the name of the material.
NormalMap
Get or set the normal map, or null if there is none.
Opacity
Get or set the opacity of the material from 0.0 for transparent to 1.0 for opaque. The default is 1.
OpacityFalloff
Get or set how opacity changes with the thickness of the object.
OpacityFalloffLevel
Get or set a multiplier against Opacity that determines how transparent an object gets at its thickest or narrowest point, as determined by OpacityFalloff. The default is 1.
SelfIlluminationColor
Get or set the color of self-illumination. The default is (1, 1, 1).
SelfIlluminationLevel
Get or set the amount of self-illumination. The default is 0.
SpecularColor
Get or set the specular color of the material, where X is red, Y is green, Z is blue, and 1.0 for each is "brightest". The default is (1, 1, 1).
SpecularLevel
Get or set a multiplier of the specular level. Values above 1 amplify the specular reflections, and below 1 darken them. The default is 1.
SpecularMap
Get or set the specular map, or null if there is none.
SpecularSoftness
Softening on specular highlights, particularly those from glancing light. The default is 0.1.
WireSize
Get or set the width of the wire to draw this material as. The default is 1.
