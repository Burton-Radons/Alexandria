ClipTerrainComponent
Perform node clipping on the terrain. Each TerrainTreeNode is sent to a clipping function which returns Containment; if it returns Contains, then the node is in view. If it returns Intersects, then the node intersects with an edge (and therefore any deeper nodes need to be tested). If it returns Disjoint, then the node is completely out of view.
