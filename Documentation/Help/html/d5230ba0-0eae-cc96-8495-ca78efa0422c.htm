IniFileBehavior Properties
IniFileBehavior Structure See Also Send Feedback

The IniFileBehavior type exposes the following members.

Properties

  Name Description
CommentDelimiter
Get or set the delimiter before a comment. The Default value is ';'; some implementations use '#'. Setting this to '\0' disables comments.
CommentDelimiter2
Sometimes ini files contain more stupid delimiters like "//". If this is null, it is not used.
CommentInValue
Get or set whether a comment delimiter in a setting value is considered to be a comment or is part of the value. The Default value is true.
Duplicates
Get behavior to follow when a duplicate setting with the same name is encountered in a file. The Default value is AllowAll.
LineContinuationEscape
Get or set whether to support line continuation escapes, which is a backslash followed by newline and causes the line break to be ignored. The Default value is false.
SettingValueDelimiter
Get or set the delimiter between a setting name and its value. The Default value is '='; some implementations use ':'.
