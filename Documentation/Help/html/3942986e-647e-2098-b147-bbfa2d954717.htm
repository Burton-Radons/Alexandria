Classes

  Class Description
PlanarTerrain
A two-dimensional terrain made up of a height map. Additional capabilities come from TerrainComponents. The PlanarTerrain is rendered as a set of blocks that are scaled to different levels based upon input from TerrainComponents that provide that information, like the [!:DistanceLodTerrainModule] and [!:OcclusionQueryTerrainModule].
PlanarTerrainBlock
A section of a PlanarTerrain.
TerrainComponent
A module that is attached to a Terrain that adds functionality to it.
TerrainEditor
Provides editing of a Terrain, using the GPU to accelerate all operations.
TerrainEditor..::..ErosionState
TerrainTreeNode
Terrain have a quadtree hierarchy for culling and LOD; this is a leaf or branch in that hierarchy.
