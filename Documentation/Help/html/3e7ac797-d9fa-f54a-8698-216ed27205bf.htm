Remarks

To implement creating, CanCreate and Create(AssetManager, Type) need to be overloaded. To implement loading, CanLoad, LoadMatch(AssetLoader), and Load(AssetLoader) need to be overloaded; LoadTypesMutable should be filled or provided in the constructor (if CanLoad is true and no types are provided in the constructor, the PrimaryType is automatically added). To implement saving, CanSave, Save(Asset, BinaryWriter), and SaveCheck(Asset) need to be overloaded; SaveTypesMutable should be filled or provided in the constructor (if CanSave is true and no types are provided in the constructor, the PrimaryType is automatically added).
