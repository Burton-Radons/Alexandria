BlendColorIndex
BlendColorIndex16
Returns a blend palette colour index, such that the first 16 colours are the normal EGA set (where a == b), and the subsequent colours are used for blending. This allows unblended colours to just use the normal palette. Spanzy!
BlendPalette
Create a blended palette, which blends colors from the perspective of either A and B or B and A, but unevenly (unless if blend is 0.5). The first colour has blend influence on the result; the second colour has (1 - blend) influence. BlendColorIndex(Int32, Int32, Int32) and BlendColorIndex16(Byte, Byte) can be used to create proper indices.
